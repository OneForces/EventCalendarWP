
# 📅 Event Calendar — плагин календаря мероприятий для WordPress

> Умный, адаптивный и красивый плагин для отображения, фильтрации и экспорта событий. Поддерживает RSVP, .ics экспорт, организаторов, локации и повторяющиеся события.

---

## 🚀 Возможности

- Кастомный тип записей **"Мероприятие"**
- Таксономии: **Тип события**, **Организатор**, **Место проведения**
- **Фильтрация по типу, дате и организатору**
- Интерактивный календарь на базе **FullCalendar**
- **Хлебные крошки**, стилизованные карточки
- Поддержка **RSVP**: форма записи на событие
- Экспорт в **.ics** файл
- **Адаптивный дизайн** и поддержка **тёмной темы**
- Совместим с WordPress 5.0+ и выше

---

## 🛠️ Установка

1. **Скачайте репозиторий**
2. Поместите папку `event-calendar` в `/wp-content/plugins/`
3. Активируйте плагин в админке WordPress → Плагины
4. Создайте новую страницу и вставьте шорткод:

```
[event_calendar]
```

---

## 🧩 Шорткоды

### 📅 Главный календарь:
```bash
[event_calendar]
```

### 📋 Список предстоящих событий:
```bash
[event_list count="5"]
```

---

## 🧑‍💻 Использование

### ➕ Добавление события:
1. В админке появится пункт **Календарь мероприятий**
2. Нажмите "Добавить новое мероприятие"
3. Укажите:
    - Название
    - Дату начала и конца
    - Повторение (если нужно)
    - Адрес и место
    - Организатора

---

## 🗺️ Кастомные таксономии

- **Организатор** — можно указать email и телефон
- **Место проведения** — можно указать адрес и описание
- Они выводятся на фронте и редактируются в админке

---

## 📤 RSVP и Экспорт

- На странице события доступна форма записи (RSVP)
- Автоматически создаётся CPT `ec_rsvp`
- Есть кнопка **📅 Экспорт в календарь (.ics)**

---

## 🎨 Стилизация

- Файл `calendar.css` содержит все стили
- Готов для светлой и тёмной темы
- Используются современные шрифты, цвета и отступы
